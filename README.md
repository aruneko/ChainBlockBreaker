# ChainBlockBreaker
## これなに
いわゆる一括採掘機能を追加する。Spigot用プラグインです。

## (動作|作成)環境
- Spigot 1.16.5
- Java 1.8
- Kotlin 1.4.30

## How To Build
以下のコマンドを入力すると、`build/libs/`以下にjarファイルができあがります。

```bash
$ ./gradlew build
```

## 使い方
スニーク状態で一括採掘機能が有効になります。以下のものが一括採掘可能です。

- 斧
    - 各種原木
    - 各種皮のむけた木
- ツルハシ
    - 石炭
    - ダイヤ
    - エメラルド
    - 金
    - 鉄
    - ラピス
    - ネザークォーツ
    - レッドストーン
    - グロウストーン
    - 古代の残骸
- シャベル
    - 粘土
    - 砂利

## 注意
- 「スニーク状態」である間だけ有効になるので、長い木を切っているときにシフトキーを離すと一括採掘状態が途中でストップします。
